## Hosting Opties

### GitHub Pages (Gratis)
1. Upload naar GitHub repository
2. Ga naar Settings > Pages
3. Selecteer "main" branch
4. App is live op: `username.github.io/pagespeed-analyzer`

### Netlify (Gratis + SSL)
1. Ga naar netlify.com
2. Sleep de folder naar de upload zone
3. Klaar! Automatisch HTTPS

### Vercel (Gratis + Zeer Snel)
```bash
npm i -g vercel
cd pagespeed-analyzer
vercel
```
